# Financial Performance Analysis - Indian IT Companies

## Overview
This project analyzes the financial performance of 5 major Indian IT companies 
(TCS, Infosys, Wipro, HCL Technologies, Tech Mahindra) over FY2021-22 to FY2025-26.

## Methods Used
- Ratio Analysis (Profitability, Liquidity, Solvency, Efficiency)
- Statistical Testing (ANOVA, Correlation)
- Composite Scoring & Ranking
- Prediction (Linear Regression)
- Clustering (K-Means)

## Files
- `Full_Analysis_Code_FINAL.py` - Complete Python analysis script
- `Master_Financial_Data.xlsx` - Raw data and calculated ratios
- `Study_Guide.md` - Detailed explanation of methodology
- `Final_Charts.png` - Visualization of results
- `Project_Summary.pdf` - Executive summary

## Tools
Python (pandas, scikit-learn, scipy), Excel, Matplotlib

## Key Findings
TCS ranked highest in overall performance, with statistically significant 
differences (ANOVA p<0.05) observed across all financial ratios.
